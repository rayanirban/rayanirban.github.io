# rayanirban.github.io

This project is a personal website that showcases various aspects including a homepage, about section, contact form, blog, and projects.

## Project Structure

- **index.html**: The main homepage of the website.
- **about.html**: Contains information about the individual or the website.
- **contact.html**: Provides a contact form for users to reach out.
- **css/styles.css**: Styles for the website, defining layout and appearance.
- **js/script.js**: JavaScript for adding interactivity to the website.
- **assets/fonts**: Directory for font files used in the website.
- **assets/icons**: Directory for icon files used in the website.
- **images/profile.svg**: Profile picture or graphic used on the website.
- **blog/index.html**: Main page for the blog section.
- **blog/posts/post-1.html**: Content for the first blog post.
- **blog/posts/post-2.html**: Content for the second blog post.
- **projects/index.html**: Showcases various projects.
- **.gitignore**: Specifies files to be ignored by version control.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the project in your preferred code editor.
3. Open `index.html` in a web browser to view the homepage.
4. Modify the content in the HTML files as needed.

## Usage Guidelines

- Update the `about.html` file with your personal information.
- Use the `contact.html` file to set up your contact form.
- Add blog posts in the `blog/posts` directory.
- Showcase your projects in the `projects/index.html` file.
